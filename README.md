This app show movies and their detail with using TMDb API.

## Getting started
- Clone 
- Get the API Key from https://developers.themoviedb.org/3/
- pod install
- build and run the project

## Dependencies
- Alamofire
- Nuke
- JEKScrollableSectionCollectionViewLayout
